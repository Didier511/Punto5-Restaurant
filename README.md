# Punto 5 React

Production-oriented React + Tailwind migration of the original Punto 5 static restaurant app.

## Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Security Direction

Do not put MongoDB credentials or admin secrets in this React app. Browser code is public after deployment. Backend secrets belong in a separate server environment; see `docs/PRODUCTION_NOTES.md`.
